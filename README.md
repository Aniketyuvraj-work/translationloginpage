Multi-Language Login Page 
A clean and modern Login Page built using HTML and CSS. The main feature of this project is the Google Translate integration, which allows users to switch the page into any language instantly.
Why I built this
I wanted to create a login UI that is both visually appealing and accessible. By using a dark overlay on a high-quality background, the form stays readable and professional.
Key Features
Language Switcher: Users can translate the entire page using the dropdown menu in the top right.
Modern Design: Uses a sleek overlay and rounded input fields for a better user experience.
Interactive UI: Includes hover effects on the login button for a more "live" feel.
Tech Used
HTML5 for the structure.
CSS3 for the styling and layout.
Google Translate API for the multi-language support.
How to Run
Download or clone this repository.
Open the index.html file in any web browser.
Select your preferred language from the menu to see it in action.
